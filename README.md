# 雙爽贏｜539 對帳列印機

這是一個純前端的 539 對帳、試算與列印工具。  
資料只存在同一台瀏覽器的本機暫存中，不登入、不上傳。重新整理或關掉重開仍會保留，按下「清空」才會清除目前明細與開獎號碼。
單價與中獎金額設定也會存在本機記憶中，方便下次開啟沿用。

## 功能

- 新增多筆明細
- 輸入號碼並保留在列印單上
- 支援坐車、立柱、連碰、手動項目
- 坐車、立柱支援上方統一設定下單價、上單價
- 連碰支援二星、三星、四星各自設定下單價、上單價
- 支援中獎金額設定：
  - 坐車：中一車獎金
  - 立柱：二星、三星、四星各自設定獎金
  - 連碰：二星、三星、四星各自設定獎金
- 自動計算：
  - 支出
  - 中獎金額
  - 小計 = 支出 - 中獎金額
  - 中間價差 = 下家小計 - 上家小計
- 支援立柱柱型，例如 `11112`
  - 柱 1：1 個號碼
  - 柱 2：1 個號碼
  - 柱 3：1 個號碼
  - 柱 4：1 個號碼
  - 柱 5：2 個號碼
- 可列印或另存 PDF

## 使用方式

直接用瀏覽器打開 `index.html`。

如果上傳到 GitHub，可以使用 GitHub Pages：

1. 建立 GitHub repository
2. 上傳本專案全部檔案
3. 到 repository 的 `Settings`
4. 進入 `Pages`
5. Source 選擇 `Deploy from a branch`
6. Branch 選擇 `main`，資料夾選擇 `/root`
7. 儲存後等待 GitHub Pages 產生網址

此工具只做試算、記帳與列印，不提供下注成立、收付款處理、派彩確認或任何交易功能。
